# KB생명 보험 데이터 스크래핑 시스템

## 개요
KB생명 홈페이지에서 보험 상품 정보(나이별, 성별별 보험료, 플랜, 특약 등)를 자동으로 수집하는 시스템입니다.

## 주요 기능
- **나이 범위**: 20세~64세 (45개 연령대)
- **성별**: 남성/여성
- **데이터 수집**: 보험료, 4개 플랜, 특약 정보, 툴팁 설명
- **웹 방화벽 우회**: 탐지 방지 기술 적용
- **자동 저장**: JSON, CSV 형태로 데이터 저장
- **에러 처리**: 실패 케이스 재시도 및 로깅

## 파일 구성

### 1. 테스트/분석 스크립트
- `kb_page_analyzer.py` - 웹페이지 구조 분석
- `kb_enhanced_scraper.py` - 방화벽 우회 테스트
- `kb_fixed_scraper.py` - 단일 케이스 테스트 (권장)

### 2. 전체 수집 스크립트
- `kb_full_scraper.py` - 전체 데이터 수집 (20-64세, 남/여)

### 3. 기존 스크립트들
- `kb_insurance_scraper.py` - 초기 버전
- `kb_scraper_main.py` - 기본 테스트 버전
- `kb_complete_scraper.py` - 상세 데이터 수집 버전

## 사용 방법

### 1. 환경 설정
```bash
pip install playwright playwright-stealth
playwright install chromium
```

### 2. 단일 테스트 실행
```bash
python kb_fixed_scraper.py
```
- 30세 남성 데이터를 수집하여 결과를 확인
- 페이지 접근 및 데이터 수집 로직 검증

### 3. 전체 데이터 수집
```bash
python kb_full_scraper.py
```
- 실행 시 전체 수집(20-64세) 또는 테스트(20-25세) 선택 가능
- 자동 저장 및 진행률 표시

## 수집되는 데이터

### 기본 정보
- 나이, 성별, 생년월일
- 수집 시간, 페이지 URL

### 보험료 정보
- 금액이 포함된 모든 텍스트
- 원, 만원 단위 데이터
- 태그명과 내용

### 플랜 정보
- 보장, 급여, 지급 관련 정보
- 플랜별 세부 내용

### 특약 정보
- 체크박스 상태 (선택/미선택)
- 특약명과 설명
- 물음표 툴팁 내용 (가능한 경우)

## 저장 파일 형식

### JSON 파일
```json
{
  "age": 30,
  "gender": "남성",
  "birthdate": "19940215",
  "premium_info": [...],
  "plan_info": [...],
  "special_clauses": [...]
}
```

### CSV 파일
나이, 성별, 생년월일, 보험료_개수, 플랜_개수, 특약_개수, 수집시간

### 1. 브라우저 설정

## 주의사항

### 1. 사용 시 주의점
- 과도한 요청으로 서버에 부하를 주지 않도록 주의
- 대기 시간을 충분히 설정
- 한 번에 많은 데이터를 수집하지 말고 단계별로 진행

### 2. 법적 고려사항
- 로봇 배제 표준(robots.txt) 확인
- 이용약관 준수
- 개인정보 보호

### 3. 기술적 제한
- 웹사이트 구조 변경 시 코드 수정 필요
- 방화벽 정책 변경 시 우회 방법 업데이트 필요
- 네트워크 환경에 따른 성능 차이

## 문제해결

### 1. 방화벽 차단
```
ERROR: firewall blocked
```
- 대기 시간 늘리기
- IP 주소 변경
- VPN 사용 고려

### 2. 타임아웃 오류
```
ERROR: timeout
```
- 네트워크 연결 확인
- timeout 값 증가
- 재시도 간격 늘리기

### 3. 데이터 누락
- 페이지 로딩 대기 시간 증가
- 셀렉터 확인 및 수정
- 디버그 모드로 확인

## 추가 기능 개발 가능 사항

### 1. 특약 자동 활성화
- 모든 특약을 체크하여 보험료 계산
- 특약별 보험료 차이 분석

### 2. 4개 플랜별 세부 분석
- 각 플랜의 상세 정보 수집
- 플랜간 비교 데이터

### 3. 물음표 툴팁 완전 수집
- 모든 도움말 정보 수집
- 특약 설명 자동 추출

### 4. 데이터 분석 기능
- 나이별 보험료 트렌드 분석
- 성별간 보험료 차이 분석
- 시각화 차트 생성
